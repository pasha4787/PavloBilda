Практична робота №1 (частина 2)

Тема: Закони розподілу та числові характеристики випадкових величин.

Варіант 2 (приклад)

Умова

Нехай X та Y — незалежні випадкові величини, рівномірно розподілені на відрізку [0,2]. Знайти закон розподілу випадкової величини Z = X + Y.

Розв’язання

Сума двох незалежних рівномірних на [0,2] має трикутну щільність на [0,4].

Функція щільності розподілу Z має вигляд:

  • fZ(z) = z/4, для 0 ≤ z < 2;
  • fZ(z) = (4 − z)/4, для 2 ≤ z ≤ 4;
  • fZ(z) = 0, поза відрізком [0,4].

Перевірка нормування:

∫[0..2] (z/4) dz + ∫[2..4] (4 − z)/4 dz = 1.
