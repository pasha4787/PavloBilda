Практична робота №3 (частина 1)

Тема: Геометрична ймовірність, аксіоматичний підхід, теореми додавання й множення.

Варіант 2 (приклад задачі)

Умова

У квадраті 0 ≤ x ≤ 1, 0 ≤ y ≤ 1 навмання обирається точка (x,y). Знайти ймовірність того, що вона потрапить у область y < 2x.

Розв’язання

Геометрична ймовірність дорівнює відношенню площі сприятливої області до площі всього квадрата.

Площа квадрата: Sкв = 1 · 1 = 1.

Площа області y < 2x в межах 0 ≤ x ≤ 1, 0 ≤ y ≤ 1 розбивається на дві частини:

  • При 0 ≤ x ≤ 0.5 маємо 0 ≤ y ≤ 2x;
  • При 0.5 ≤ x ≤ 1 маємо 0 ≤ y ≤ 1 (бо 2x > 1).

Обчислюємо площу:

S1 = ∫[0..0.5] 2x dx = 2 · (0.5² / 2) = 0.500,

S2 = ∫[0.5..1] 1 dx = 1 · (1 − 0.5) = 0.500.

Загальна площа сприятливої області:

S = S1 + S2 = 1.000.

Оскільки площа всього простору подій дорівнює 1, то ймовірність P(y < 2x) = S = 1.000.
