Практична робота №3 (частина 2)

Тема: Найпростіший потік, елементи теорії СМО, ланцюги Маркова.

Варіант 2 (приклад задачі про ланцюг Маркова)

Умова

Нехай маємо однорідний ланцюг Маркова з двома станами 1 та 2 та матрицею переходів за один крок:

P = [[0.4, 0.6], [0.3, 0.7]].

Знайти матрицю переходів за два кроки P².

Розв’язання

Матриця переходів за два кроки — це добуток матриці P самої на себе:

P² = P · P.

Обчислюємо (покроково):

  • P²(1,1) = 0.4·0.4 + 0.6·0.3 = 0.16 + 0.18 = 0.34;
  • P²(1,2) = 0.4·0.6 + 0.6·0.7 = 0.24 + 0.42 = 0.66;
  • P²(2,1) = 0.3·0.4 + 0.7·0.3 = 0.12 + 0.21 = 0.33;
  • P²(2,2) = 0.3·0.6 + 0.7·0.7 = 0.18 + 0.49 = 0.67.

Отже,

P² = [[0.34, 0.66], [0.33, 0.67]].
